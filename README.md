# First Semester Network Security Project (Cisco Packet Tracer)

## Project Overview
This project was completed as part of my first semester coursework.
It simulates a secure small enterprise network for a financial services company,
focusing on network segmentation ,controlled inter-VLAN communication,
and basic security enforcement.

## Network Departments
- Admin (VLAN 10)
- Sales (VLAN 20)
- IT Support (VLAN 30)

## Key Concepts Implemented
- VLAN segmentation
- Inter-VLAN routing (Router-on-a-Stick)
- Access Control Lists (ACLs)
- Basic network security principles

## Tools Used
- Cisco Packet Tracer
- Networking fundamentals
